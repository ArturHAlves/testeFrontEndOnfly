# testeFrontOnFly

## Stack

- Vue 3 + TypeScript
- Vite
- Quasar V2
- Sass (SCSS)
- BEM para nomenclatura CSS

## Arquitetura

```text
src/
  assets/
    styles/
  composables/
  constants/
  layouts/
  pages/
  plugins/
  router/
  main.ts
```

## Comandos

```sh
npm install
npm run dev
npm run build
```
